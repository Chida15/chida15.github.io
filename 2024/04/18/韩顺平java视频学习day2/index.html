数据类型

（1）数据类型分为基本数据类型和引用数据类型（类class、接口interface和数组[]）

（2）Java的整数类型有固定的范围和字段长度，不受具体os的影响以保证Java程序的可移植性。而C++的整数类型大小取决于编译器实现和运行的操作系统

（3）Java的整型常量默认为int型，声明long型常量须后加’ L’，浮点型常量默认为double型，声明float型常量须后加’f’或’F’
